# Sample ERC721 Project

This project demonstrates a sample ERC721 smart contract. It comes with tests for that contract and a script that deploys that contract.

### Compile Project  
```shell
npx hardhat compile
```  

### Run Tests  
```shell
npx hardhat test
```

### Deploy Contract  
```shell
npx hardhat run scripts/deploy.ts
```
